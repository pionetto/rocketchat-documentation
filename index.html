Introdução

O Rocket.chat é a solução para melhorar a produtividade da Prefeitura Municipal de Santarém, garantindo uma comunicação ágil, segura e em tempo real. A plataforma é muito versátil com recursos como compartilhamento de arquivos e chamadas de videoconferência. Além de otimizar a comunicação interna, é possível melhorar a comunicação com a população, através de um Livechat e a possibilidade de criação de um canal de comunicação. Em tempos de Pandemia e a necessidade de se evitar aglomeração, esta plataforma torna-se necessária.

Instalação do RocketChat

A instalação do rocketchat pode ser realizada de várias formas, e constam no repositório da empresa. Dentre as instalações, podemos citar:

Docker

Implantar com docker compose

OU Use a imagem de compilação automatizada do lançamento mais recente

docker pull rocketchat/rocket.chat:latest

OU selecione uma versão específica (detalhes dos lançamentos disponíveis):

docker pull rocketchat/rocket.chat:vX.X.X

OU a imagem de registro oficial do docker, contendo a versão estável recente aprovada pelo Docker:

docker pull rocket.chat Snap sudo snap install rocketchat-server

Após isso, é necessário abrir o link http://localhost:3000 em seu browser. Serão solicitados alguns dados para registro e configurações iniciais do servidor RocketChat, como e-mail, usuário e senha para acesso. Após esta etapa, você estará logado em uma tela do ambiente do usuário Administrador, que será descrita na próxima sessão deste documento.
